Sobre o sistema

O Easy Help é um MVP acadêmico desenvolvido para demonstrar a separação entre frontend e backend, com consumo de API REST em Flask.

Além disso, o sistema possui um modo offline com simulação de API em memória, permitindo a execução completa sem backend, ideal para demonstração.

Tecnologias utilizadas

  • HTML5
  • CSS3
  • JavaScript
  • Bootstrap
  • Python com Flask
  • SQLite
  • Swagger / OpenAPI

Fluxo do sistema

  • Cadastrar departamento
  • Cadastrar usuário vinculado ao departamento
  • Cadastrar chamado vinculado ao usuário
  • Acompanhar andamento e atualizar status
  • Resolver, reabrir ou fechar atendimento

Objetivo do MVP

Demonstrar uma SPA funcional, com integração real com API, organização visual, interatividade e aderência aos requisitos da disciplina.
